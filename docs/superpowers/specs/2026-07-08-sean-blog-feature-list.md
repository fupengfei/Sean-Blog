# Sean's AI World — 功能清单

> 版本: v1.0 | 日期: 2026-07-08

---

## 1. 首页

| 编号 | 功能 | 说明 |
|------|------|------|
| 1.1 | 精选项目展示 | 从项目库中选取 `is_featured=true` 的项目，数量少时有占位排版 |
| 1.2 | 精选文章展示 | 从文章库中选取 `is_featured=true` 的文章，用设计稿中的卡片样式 |
| 1.3 | CTA 底部栏 | 标题 + 描述文案，两个操作按钮 |

### CTA 底部栏两个功能

| 编号 | 功能 | 说明 |
|------|------|------|
| 1.3.1 | 发送邮件 | 点击后调用本地邮件客户端打开（`mailto:` 链接），后台记录请求日志 |
| 1.3.2 | 获取简历 | 弹出表单：公司名称 + 邮箱地址，提交后后台记录 |

---

## 2. 博客模块

| 编号 | 功能 | 说明 |
|------|------|------|
| 2.1 | 文章列表页 | 支持按分类筛选、按标签筛选、按时间排序 |
| 2.2 | 文章详情页 | MD 渲染展示，文章标题、时间、分类、标签、阅读量 |
| 2.3 | Skill 目录浏览 | 展示通过 Admin 上传的文件目录树，类似 GitHub 文件浏览器，支持展开/收起目录、查看文件内容 |

---

## 3. 项目模块

| 编号 | 功能 | 说明 |
|------|------|------|
| 3.1 | 项目列表页 | 卡片网格展示，项目截图、标题、描述、标签、跳转链接 |
| 3.2 | 项目跳转 | 点击跳转到配置的外部 URL（或 GitHub 仓库链接） |

---

## 4. 关于我

| 编号 | 功能 | 说明 |
|------|------|------|
| 4.1 | 个人介绍 | 文字介绍区域，可展示技术栈、经验年限 |
| 4.2 | 联系方式 | 邮箱、位置等联系信息展示 |

---

## 5. Admin 管理台

### 5.1 文章管理

| 编号 | 功能 | 说明 |
|------|------|------|
| 5.1.1 | 文章列表 | 表格展示：标题、分类、状态、创建时间、操作 |
| 5.1.2 | 新建/编辑文章 | MD 文件直接上传，后端解析渲染为 HTML，提取标题/excerpt，支持设置分类和标签 |
| 5.1.3 | 删除文章 | 软删除/回收站 |
| 5.1.4 | 精选标记 | 勾选文章为精选，显示在首页 |
| 5.1.5 | 分类管理 | 增删改分类 |
| 5.1.6 | 标签管理 | 增删改标签 |

### 5.2 项目管理

| 编号 | 功能 | 说明 |
|------|------|------|
| 5.2.1 | 项目列表 | 表格展示：标题、URL、状态、排序 |
| 5.2.2 | 新建/编辑项目 | 表单：标题、描述、URL、GitHub URL、封面图上传、标签设置 |
| 5.2.3 | 删除项目 | 删除 |
| 5.2.4 | 精选标记 | 标记显示在首页 |
| 5.2.5 | 排序 | 拖拽或上下移动调整展示顺序 |

### 5.3 文件目录管理（Skill Bundle）

| 编号 | 功能 | 说明 |
|------|------|------|
| 5.3.1 | Bundle 列表 | 展示所有上传的文件目录包 |
| 5.3.2 | 上传 Bundle | zip 文件上传，后端自动解压，解析目录树存入数据库 |
| 5.3.3 | 删除 Bundle | 删除整个文件目录及所有文件 |
| 5.3.4 | 发布/取消发布 | 控制是否在博客页面展示 |
| 5.3.5 | 目录树预览 | Admin 内预览目录结构效果 |

### 5.4 联系记录

| 编号 | 功能 | 说明 |
|------|------|------|
| 5.4.1 | 邮件请求记录 | 列表展示：请求时间、IP |
| 5.4.2 | 简历请求记录 | 列表展示：公司名称、邮箱、请求时间、IP |

### 5.5 认证

| 编号 | 功能 | 说明 |
|------|------|------|
| 5.5.1 | 登录 | 用户名 + 密码，JWT Token |
| 5.5.2 | Token 过期处理 | 过期后跳转登录页 |
| 5.5.3 | 修改密码 | Admin 内改密 |
